1.HelloWorld 를 출력하는 프로그램의 과정을 설명하시오.
확장자가 java인 파일에 소스코드를 작성하여 javac 명령어로 컴파일후 만들어진 class파일을 java 명령어로 실행

2.아래의 명령어를 설명하시오(뭐하는건가).
-javac
확장자가 java인 파일을 컴파일하여 class파일을 만들어주는 명령어
-java
javac로 컴파일하여 생성된 class파일을 실행시켜주는 역할

3.컴파일이란 무엇인가요?
프로그래머가 작성한 코드를 2진수밖에 모르는 cpu가 알아들을 수 있게 2진수로 번역해주는 번역기 역할

4.java 언어를 창시한 사람은?
제임스 고슬링

5.JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
java development kit, 자바를 개발하기위한 툴킷, 오라클,
자바를 개발하기 위해 여러 기능을 제공하고 있는 패키지

6.JVM 무엇의 약자이며,이란 무엇인가?
java virtual machine, java 코드를 os에 맞게 해석해주는 역할
각 os에 맞는 jvm을 설치하면 어느 os기반에서 작성한 코드인지 상관없이 윈도우,맥,리눅스에서 같은 코드를 실행 가능
